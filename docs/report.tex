\section{Introduction}
% ======================================================================

\par
This project explores the relationship between state firearm legislation and
gun violence in the United States using data spanning 1976--2023. While
estimated household gun ownership is included as a contextual variable, it is
not treated as the primary explanatory factor. Evidence from countries such as
Finland and Switzerland where high gun ownership coexists with low rates of gun
violence, suggests that the regulatory environment surrounding firearms may
matter more than ownership prevalence alone. This project therefore focuses on
whether the strength and type of state firearm laws is associated with lower
rates of firearm homicide, and whether changes in legislation over time
correspond to measurable shifts in a state's gun violence standing.

\subsection{Methodology}

\par
The project followed the process below to achieve the goal described above:
\begin{enumerate}
	\item Dataset selection and join strategy design
	\item Data analysis outside Tableau
	\item Research questions derived from data exploration
	\item Visualization types needed to answer each question and storytell
	\item Data source connection, relationships, and calculated fields in Tableau
	\item Tableau feature selection (parameters, sets, filters, actions, tooltips)
	\item Dashboard design and interactive storytelling
	\item Use of AI assistance for laborious or repetitive tasks
	      (see Section~\ref{sec:ai})
\end{enumerate}

\subsection{Sources}

\par
Numerous sources are used throughout the project such as:

\begin{itemize}
	\item Official dataset documentation and
	      codebooks~\autocite{ownershipproxy,statefirearmlaw}
	\item Harvard Dataverse and Tufts CTSI dataset
	      metadata~\autocite{kang2023,statefirearmlaw}
	\item Academic literature on firearm ownership
	      proxies~\autocite{extendingfss,kang2023}
	\item Academic literature on gun law
	      effects~\autocite{kopernij1999,koper2004,donohue2022}
	\item Tableau official documentation~\autocite{tableau}
	\item Lecture notebooks and course materials
	\item AI assistance (declared in Section~\ref{sec:ai})
\end{itemize}

Regardless of the source, these are indicated through bibliography and

\section{Datasets}\label{sec:datasets}
% ======================================================================

\par
Two public datasets were selected and used in conjunction for this project.
Both are accessible through their corresponding links and are available in
different formats. The first dataset was identified independently; finding a
compatible second dataset was AI-assisted (Claude Sonnet 4.6;
see Section~\ref{sec:ai}).

\begin{quote}
\textbf{Prompt:} \textit{I already have a state-year level dataset on
estimated household gun ownership in the US. I need a second dataset that
contains information about firearm legislation on a state-year basis so I
can join them easily. What publicly available datasets would you recommend?}
\end{quote}

\begin{quote}
\textbf{Response summary:} Recommended the Tufts CTSI State Firearm Law
Database, noting that it tracks 72 binary law provisions per U.S.\ state
per year from 1976 to 2024, shares the same state--year grain as the
ownership proxy dataset, and is widely used in public health research,
making it a straightforward join candidate.
\end{quote}

\subsection{Dataset 1: Firearm Suicide Proxy for Household Gun Ownership, 1949--2023}

\par
The first dataset\autocite{ownershipproxy} is a state-year level proxy for
household gun ownership in the United States. Direct gun ownership data is
not publicly available at the state level, so this dataset uses the firearm
suicide proxy methodology: the proportion of suicides committed with a firearm
(FSS) is used as an indirect estimate of household gun prevalence by state and
year. It is hosted on Harvard Dataverse.

\begin{itemize}
	\item \textbf{Source}: Harvard Dataverse, Kang et al.\autocite{kang2023}
	\item \textbf{Grain}: One row per U.S. state per year
	\item \textbf{Key columns}: State, Year, FSS (firearm suicide share),
	      firearm suicides, total suicides, firearm homicide rate,
	      total homicides, population
	\item \textbf{Date span}: 1949--2023 (75 years, 50 states + D.C.)
	\item \textbf{Format}: Comma-separated values (\texttt{.csv})
\end{itemize}

\par
This dataset was chosen because it provides a long term, population adjusted,
and academically validated measure of gun prevalence that can be compared
across states and over time. FSS is not a literal count of guns per household
but an indirect proxy widely used in public health and policy
research\autocite{extendingfss}. FSS is retained in this project as a
contextual indicator of estimated gun prevalence on the scatter plot;
firearm homicide rate serves as the primary outcome measure throughout.

\subsection{Dataset 2: State Firearm Law Database, 1976--2024}

\par
The second dataset\autocite{statefirearmlaw} is maintained by the State
Firearm Law Research Group at Tufts University and tracks the presence or
absence of 72 firearm safety law provisions across all 50 U.S. states for
every year from 1976 to 2024.

\begin{itemize}
	\item \textbf{Source}: Tufts CTSI --- State Firearm Law
	      Database\autocite{statefirearmlaw}
	\item \textbf{Grain}: One row per U.S. state per year
	\item \textbf{Key columns}: State, Year, 72 binary law indicator columns
	      (e.g.\ permit required, background check, waiting period,
	      stand-your-ground, child access prevention)
	\item \textbf{Date span}: 1976--2024
	\item \textbf{Format}: Excel (\texttt{.xlsx})
\end{itemize}

\par
This dataset was chosen because it provides a historically accurate,
year-by-year record of state firearm legislation, enabling analysis of
whether law changes correspond to changes in gun violence rates. It is
non partisan by design and is widely used in academic public health
research\autocite{statefirearmlaw}.

\subsection{Join Strategy}

\par
Both datasets were connected in Tableau using a \textbf{relationship} on the
shared keys \texttt{state} and \texttt{year}, configured with
\textbf{many-to-many} cardinality and \textbf{some records match} referential
integrity on both sides. Many-to-many cardinality is appropriate here because
both tables share the same grain (one row per state per year) and neither side
acts as a unique dimension table. The ``some records match'' setting instructs
Tableau not to assume all keys are present in both tables, handling
non-overlapping rows safely without discarding data prematurely. Tableau
resolves the relationship by performing aggregations on each table
independently before combining results, preventing row duplication. The
overlapping date range of both datasets yields an effective analysis window
of \textbf{1976--2023}.

\subsection{Analysis}

\par
Prior to connecting the data in Tableau, both files were inspected to
understand structure, null patterns, and label consistency.
Key observations:

\begin{itemize}
	\item Both datasets use identical full state name format
	      (e.g.\ \texttt{Alabama}, \texttt{North Carolina}),
	      allowing for a join on state
	\item The ownership proxy file includes several pre calculated rate
	      columns derived from raw counts and population
	\item The law database contains 72 binary columns but it also includes
	      a total law count column (\texttt{lawtotal}) that can be used directly
	\item Both datasets include a sufficient overlapping year window between
	      1976--2023 that can be used to join
\end{itemize}

\subsection{Processing}

\par
Both source files were connected to Tableau without modification to the raw
data. No column renaming, deduplication, or external pre-processing was
required. The only structural adjustment was the exclusion of the District of
Columbia via a Tableau filter, as it is present in the ownership proxy dataset
but absent from the state firearm laws dataset, which would have prevented a
valid join for that entity. All visualizations are filtered to the years
1976--2023, the window for which both datasets have complete data.

\subsection{Research Questions}

\par
Three questions were formulated based on data exploration and storytelling intent:

\begin{enumerate}
	\item Is a state's firearm homicide rate associated with the number
	      and type of firearm laws it has in place?\sref{q1}
	\item How have firearm homicide rates evolved over time depending on
	      the firearm laws passed in each state, whether measured by
	      the quantity of laws or by the categories of law types enacted?\sref{q2}
	\item For a specific state, does passing more firearm laws correspond
	      to a reduction in firearm homicide rates, and how does this
	      affect its ranking relative to other states?\sref{q3}
\end{enumerate}

\subsubsection{Question 1: Homicide Rates and the Law Landscape}\label{q1}

\par
This is the entry point of the analysis. Before drawing any temporal or causal
conclusions, it is necessary to establish whether a cross-sectional pattern
exists: do states that have enacted more laws or certain types of laws tend to
have lower firearm homicide rates? FSS is included as a contextual variable
to show where estimated gun ownership sits in this picture, without treating
it as an outcome or a direct cause.

\subsubsection{Question 2: Homicide Trends by Law Profile Over Time}\label{q2}

\par
If a cross-sectional association exists, the natural follow-up is whether it
has strengthened or weakened over time. By grouping states according to their
law profile --- either by quantity tier or by law type category
(see Section~\ref{sec:lawcat}) --- it becomes possible to observe whether
homicide trajectories have diverged, and whether that divergence accelerated
after key legislative periods such as the early 1990s.

\subsubsection{Question 3: State-Level Law History and Homicide Standing}\label{q3}

\par
Q1 and Q2 analyze the broader picture; Q3 brings the analysis down to the
individual state, making it personally relevant and explorable for any viewer.
The motivation is to let someone examine their own state: when did it pass
laws, how many, and does its current homicide ranking reflect that legislative
history?

\section{Visualization Prep}
% ======================================================================

\par
Before building charts in Tableau, design and structural decisions were made
to align visualizations with the storytelling intent and assignment
requirements.

% -----------------------------------------------------------------------
\subsection{Storytelling}
% -----------------------------------------------------------------------

\par
The project follows a four-act structure. Act~1 establishes the geographic
and cross-sectional baseline (where is violence highest; what does the law
landscape look like). Act~2 adds the temporal dimension, showing how homicide
rates have diverged by law profile over 1976--2023, using both categorization
schemes defined in Section~\ref{sec:lawcat}: quantity tier and law type group
(Popular vs.\ Evidence-Based). Act~3 narrows to individual states, tracing
each state's legislative history against its homicide ranking relative to all
others. Act~4 zooms into the selected law provisions directly, comparing the
average homicide trend for the Popular and Evidence-Based groups to assess
which shows the more consistent association with changes in homicide rates.

% -----------------------------------------------------------------------
\subsection{Law Categorization}\label{sec:lawcat}
% -----------------------------------------------------------------------

\par
Two complementary categorization schemes are used across the visualizations
to group states by their law profile. Both are applied through a shared
parameter that lets the viewer toggle between them.

\par
\textbf{By quantity} --- states are assigned to one of three tiers based on
their total law count (\texttt{lawtotal}) for a given year:

\begin{itemize}
	\item \textbf{Low} --- states with a relatively small number of laws in
	      effect
	\item \textbf{Medium} --- states in the middle tier
	\item \textbf{High} --- states with a large number of laws in effect
\end{itemize}

\par
\textbf{By type} --- rather than comparing all 72 laws, a focused subset was
selected: the three most widely adopted laws (``Popular'') and the three laws
with the strongest evidence base in the literature (``Evidence-Based''). States
are then classified into four mutually exclusive categories based on whether
they have enacted two or more laws from either group:

\begin{itemize}
	\item \textbf{Weak} --- fewer than two Popular laws and fewer than two
	      Evidence-Based laws
	\item \textbf{Popular} --- two or more Popular laws, but fewer than two
	      Evidence-Based laws
	\item \textbf{Evidence} --- two or more Evidence-Based laws, but fewer than
	      two Popular laws
	\item \textbf{Both} --- two or more laws from each group
\end{itemize}

% -----------------------------------------------------------------------
\subsection{Colors and Themes}\label{colors}
% -----------------------------------------------------------------------

\par
Color choices were made deliberately to minimize viewer bias and maximize
clarity:

\begin{itemize}
	\item \textbf{Neutral colors} were used throughout to avoid leading the
	      viewer toward a political interpretation. Grey tones were applied
	      to states with no meaningful change in laws or with a weak law
	      profile, so that they recede visually without being misleading.
	\item \textbf{Orange gradient} --- a somewhat neutral but gravity-suggesting
	      tone --- was chosen for value-encoding gradients (e.g.\ firearm
	      homicide rates on maps and scatter plots), where darker shades
	      indicate higher rates.
	\item \textbf{Gridlines} were kept minimal to reduce visual noise, with
	      color used as the primary channel for directing the viewer's
	      attention to data marks rather than chart furniture.
\end{itemize}

\subsection{Q1 -- Homicide Rates and the Law Landscape}\label{sec:viz:q1}

\par
Two charts answer Q1\sref{q1}. The first is a \textbf{choropleth map}
(Figure~\ref{fig:q1chart1}) displaying the firearm homicide rate across all
U.S.\ states for a user-selected year, encoded with an orange gradient so
darker states indicate higher rates. A \textbf{parameter} drives the year
selection, and a \textbf{calculated field} supplies the two-letter
abbreviation labels shown on each state mark.

\par
Generating the state abbreviation calculated field was AI-assisted
(Claude Sonnet 4.6; see Section~\ref{sec:ai}), as writing all 50
\texttt{WHEN} clauses by hand is purely mechanical and error-prone.
The prompt and response are shown below (Prompt~\ref{prompt:abbr}).

\begin{quote}\label{prompt:abbr}
\textbf{Prompt:} \textit{Generate a Tableau calculated field using a CASE
statement that maps each U.S.\ state's full name to its two-letter
abbreviation for all 50 states. Only output the code.}
\end{quote}

\begin{quote}
\textbf{Response:}
\begin{Verbatim}[fontsize=\footnotesize]
CASE [State]
  WHEN "Alabama" THEN "AL"
  WHEN "Alaska"  THEN "AK"
  ...
  WHEN "Wisconsin" THEN "WI"
  WHEN "Wyoming"   THEN "WY"
END
\end{Verbatim}
\end{quote}

The second chart (Figure~\ref{fig:q1chart2}) is a \textbf{scatter plot}

\subsection{Q3 -- State-Level Law History and Homicide Standing}\label{sec:viz:q3}

\par
Two charts answer Q3\sref{q3}. The first (Figure~\ref{fig:q3chart1}) is a

% ======================================================================
\section{Conclusions}
% ======================================================================

\par
Visualizations reveal the following:

\begin{itemize}
	\item Q1\sref{q1}: States with fewer firearm laws consistently show higher
	      firearm homicide rates, both geographically and in the cross-sectional
	      scatter view (Figure~\ref{fig:q1chart2}).
	\item Q2\sref{q2}: States with more laws, whether measured by quantity or
	      type category, have seen steeper declines in firearm homicide since
	      the 1990s (Figure~\ref{fig:q2chart}).
	\item Q3\sref{q3}: In several states, sustained legislative growth coincides
	      with a visible improvement in homicide ranking relative to other
	      states, though no causal claim is made
	      (Figures~\ref{fig:q3chart1} and~\ref{fig:q3chart2}).
	\item Gun ownership (FSS) provides useful contextual framing but is
	      not treated as a causal variable; the regulatory environment
	      is the primary analytical lens throughout the project.
\end{itemize}

\par
Limitations include the proxy nature of the FSS ownership estimate,
the absence of statistical controls for confounding variables such as
poverty and urbanization, and the use of a static join on state name
which may mask within-state variation.
