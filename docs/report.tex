\section{Introduction}

\par
This project explores the relationship between state firearm legislation and
gun violence in the United States, using data spanning 1949-2023. While
estimated gun ownership is included as a contextual variable, it is not
treated as the primary explanatory factor. Evidence from countries such as
Finland and Switzerland, where high gun ownership coexists with low rates of
gun violence, suggests that the regulatory environment surrounding firearms may
matter moer than ownership prevalence alone. This project therefore focuses on
whether the strength of state firearm laws is associated with lower rates of
firearm homicide and suicide, and whether changes in legislation over time
correspond to measurable shifts in gun violence rates.

\subsection{Methodology}

\par
The project followed the process below to achieve the goal described above:
\begin{enumerate}
	\item Dataset selection and join strategy design
	\item Data analysis and cleaning outside Tableau
	\item Research questions derived from data exploration
	\item Visualization types needed to answer each question and storytell
	\item Data source connection, relationships, and calculated fields in Tableau
	\item Tableau feature selection (parameters, filters, actions, tooltips)
	\item Dashboard design and interactive storytelling
\end{enumerate}

\subsection{Sources}

\par
Numerous sources are used throughout the project such as:

\begin{itemize}
	\item Official dataset documentation and codebooks
	\item ATF and Harvard Dataverse dataset metadata
	\item Academic literature on firearm ownership proxies
	\item Tableau official documentation
	\item Lecture notebooks and course materials
	\item AI assistance (declared below)
\end{itemize}

\par
Regardless of the source, these are indicated through bibliography and
references, or in the case of AI, an explicit note indicating the model and
prompt used.

\subsection{AI Usage}

\par
AI was used during this project in a critical and supervised manner. All
AI-generated suggestions were reviewed and scrutinized before being applied,
with particular attention to the risk of hallucination. AI was consulted
primarily for dataset pairing strategy, question framing, and Tableau workflow
guidance, and was not used to generate data or conclusions directly.

\section{Datasets}

\par
Two public datasets were selected and joined for this project. Both were
required to be different from those used in the Python assignment, and had not
been claimed by another student.

\subsection{Dataset 1: State-Level Household Gun Ownership Proxy, 1949--2020}

\par
The first dataset\autocite{ownershipproxy} is a state-year level proxy for
household gun ownership in the United States, covering 1949 to 2020. It is
hosted on Harvard Dataverse and is derived from the firearm suicide proxy
methodology: the proportion of suicides committed with a firearm is used as
an indirect estimate of household gun prevalence by state and year.

\begin{itemize}
	\item \textbf{Source}: Harvard Dataverse, Kang et al.\autocite{kang2023}
	\item \textbf{Grain}: One row per U.S. state per year
	\item \textbf{Key columns}: State, Year, Firearm suicide proxy (FSS),
	      total suicides, firearm suicides, population
	\item \textbf{Date span}: 1949--2020 (72 years, 50 states + D.C.)
	\item \textbf{Format}: CSV
\end{itemize}

\par
This dataset was chosen because it provides a long-term, population-adjusted,
and academically validated measure of gun prevalence that can be compared
across states and over time. It is not a literal count of guns per household
but an indirect proxy widely used in public health and policy
research\autocite{extendingfss}.

\subsection{Dataset 2: ATF Gun Traces by State and Firearm Type, 2014--2023}

\par
The second dataset\autocite{atftraces} is published by The Trace via the Gun
Violence Data Hub and originates from the U.S. Bureau of Alcohol, Tobacco,
Firearms and Explosives (ATF). It aggregates firearm trace requests submitted
by law enforcement, broken down by state, year, and firearm type.

\begin{itemize}
	\item \textbf{Source}: The Trace / ATF eTrace system\autocite{atfguide}
	\item \textbf{Grain}: One row per state per year per firearm type
	\item \textbf{Key columns}: State, Year, Firearm type (pistol, revolver,
	      rifle, shotgun, other), Trace count
	\item \textbf{Date span}: 2014--2023
	\item \textbf{Format}: CSV (\texttt{types\_by\_state.csv})
\end{itemize}

\par
This dataset was chosen because it provides a direct state-level breakdown of
what types of crime-recovered firearms are most commonly traced by law
enforcement. It complements the ownership proxy by offering a second lens on
firearm patterns: not how many households likely have guns, but what kinds of
guns appear in traced-crime contexts and how that varies by
state\autocite{atftraces}.

\subsection{Join Strategy}

\par
Both datasets were aggregated and joined on the shared keys \texttt{state} and
\texttt{year}. Since the ownership proxy spans 1949--2020 and the ATF trace
data spans 2014--2023, the effective overlap window is \textbf{2014--2020}.
The join was performed in Tableau using a relationship between the two data
sources on the matching state-year grain, ensuring the workbook remains
reproducible and reviewable.

\subsection{Analysis}

\par
Prior to connecting the data in Tableau, both files were inspected to
understand structure, null patterns, and label consistency.
Key observations:

\begin{itemize}
	\item State names required standardisation across both files (full names
	      vs. abbreviations)
	\item The ownership proxy file includes several calculated rate columns
	      derived from raw counts and population
	\item The ATF trace file contains one row per state-year-type combination,
	      requiring aggregation before the join
	\item The overlapping year range (2014--2020) was confirmed as sufficient
	      for temporal analysis and question answering
\end{itemize}

\subsection{Processing}

\par
Key processing steps applied before connecting to Tableau:

\begin{itemize}
	\item Standardised state name format across both files to enable the join
	\item Filtered both files to the 2014--2020 overlap window
	\item Aggregated ATF trace file to one row per state-year with total
	      trace count and per-type share columns
	\item Retained population-adjusted rates from the ownership proxy
	      for per-capita analysis in Tableau
	\item Removed U.S. territory rows (Puerto Rico, Guam, etc.) to keep
	      the analysis to the 50 states and D.C.
\end{itemize}

\subsection{Research Questions}

Three questions were formulated based on data exploration and storytelling intent:

\begin{enumerate}
	\item How has estimated household gun prevalence varied by state over time?
	\item What firearm types dominate ATF trace summaries by state?
	\item Do states with higher estimated gun prevalence show different traced
	      firearm type patterns?
\end{enumerate}

\subsubsection{Question 1: Household Gun Prevalence Over Time}\label{q1}

\par
Question 1 establishes the baseline of the story. By mapping and trending the
gun ownership proxy across states and years, the viewer understands where guns
are most prevalent and whether that has changed over time. This provides
essential context for interpreting the ATF trace patterns addressed in the
following two questions.

\subsubsection{Question 2: Traced Firearm Type Composition}\label{q2}

\par
Question 2 shifts the lens from prevalence to composition. By exploring which
firearm types appear most frequently in ATF trace data by state, the viewer
can identify regional patterns in traced crime guns and assess whether certain
states are more associated with specific weapon types than others.

\subsubsection{Question 3: Prevalence vs.\ Traced Patterns}\label{q3}

\par
Question 3 connects both datasets analytically. It asks whether states with
higher estimated household gun prevalence also exhibit different patterns in
ATF-traced firearm types. This is framed as an association analysis rather
than a causal claim, consistent with the observational nature of both
datasets and standard practice in firearm research\autocite{extendingfss}.

\section{Visualization Prep}

\par
Before building charts in Tableau, design and structural decisions were made
to align visualizations with the storytelling intent and assignment
requirements.

\subsection{Storytelling}

\par
The project follows a three-act structure. Q1\sref{q1} opens with the
geographic and temporal baseline: where gun prevalence is highest and how it
has evolved. Q2\sref{q2} introduces the second dimension: what kinds of
firearms are being traced by state. Q3\sref{q3} closes with an analytical
comparison between the two datasets, asking whether prevalence and type
patterns are associated. The viewer moves from observation to exploration to
interpretation.

\subsection{Colors and Themes}\label{colors}

\par
Colors were chosen to encode meaning consistently, directly addressing the
feedback from the previous assignment that color was under-used as a visual
encoding mechanism:

\begin{itemize}
	\item A \textbf{sequential red palette} was used for ownership proxy maps,
	      where darker shades indicate higher estimated gun prevalence.
	\item \textbf{Categorical colors} were used for firearm types (pistol,
	      revolver, rifle, shotgun), kept consistent across all sheets so
	      the same type always appears in the same color.
	\item A \textbf{diverging palette} distinguishes states above and below
	      the national average in scatter and detail views.
	\item Backgrounds are light and gridlines are minimal to reduce visual
	      noise and keep focus on data marks.
\end{itemize}

\par
These decisions were made in line with visualization theory from Unit 1 and
with attention to color-blindness accessibility by avoiding red-green
contrasts as the sole distinguishing factor.

\subsection{Tableau Feature Choices}

\par
The following Tableau features were planned and used across this project:

\begin{itemize}
	\item \textbf{Calculated fields}: Gun prevalence quintile, trace rate
	      per 100k population, firearm type share percentage,
	      above/below national average flag
	\item \textbf{Parameters}: Year selector allowing the viewer to update
	      all views simultaneously to a chosen year
	\item \textbf{Filters}: State multiselect filter, year range slider,
	      firearm type selector
	\item \textbf{Dashboard actions}: Click-to-filter from the map to the
	      bar chart and scatter plot
	\item \textbf{Enhanced tooltips}: Display state, year, ownership proxy
	      rate, total trace count, and top firearm type per state
	\item \textbf{Reference lines}: National average line in trend and
	      scatter views
\end{itemize}

\section{Visualizations}

\par
A consistent set of calculated fields and a shared color scheme were applied
across all visualizations. Field names are self-explanatory (e.g.\
\texttt{Ownership Proxy Rate}, \texttt{Trace Rate per 100k},
\texttt{Rifle Share \%}) to ensure readability and reproducibility.

\subsection{Q1 -- Household Gun Prevalence by State}

\par
Two visualizations answer Question 1\sref{q1}: a filled map showing geographic
distribution of the ownership proxy and a line chart showing how the proxy has
evolved over the available time range.

\par
The \textbf{filled map} was chosen because the proxy is a state-level measure
and geography is the most natural encoding for state-level comparisons.
A sequential red palette was applied and a year parameter allows the viewer
to explore any year in the 2014--2020 window.

\begin{figure}[h]
	\centering
	\includegraphics[width=\textwidth]{../figures/q1_map.png}
	\caption{Q1 Chart -- U.S. State Gun Ownership Proxy Map}
	\label{fig:q1map}
\end{figure}

\par
The \textbf{line chart} places Year on Columns and the national average proxy
on Rows, with individual state lines in light grey and the national average
highlighted. A reference line marks the overall mean.

\begin{figure}[h]
	\centering
	\includegraphics[width=\textwidth]{../figures/q1_trend.png}
	\caption{Q1 Chart -- Gun Ownership Proxy Trend Over Time}
	\label{fig:q1trend}
\end{figure}

\textbf{Insights}: Both views reveal strong regional patterns. States in the
South and Mountain West consistently show higher proxy values, while states in
the Northeast show lower values. The national trend shows a gradual decline
in the proxy from the mid-1990s to 2020, visible within the 2014--2020
analysis window.

\subsection{Q2 -- Firearm Type Composition by State}

\par
A \textbf{stacked bar chart} was used to answer Question 2\sref{q2}. Each bar
represents one state, segmented by firearm type share. Categorical colors
encode each type consistently. The stacked bar was preferred over a pie chart
or treemap because it allows direct cross-state comparison along a shared axis.

\begin{figure}[h]
	\centering
	\includegraphics[width=\textwidth]{../figures/q2_stacked_bar.png}
	\caption{Q2 Chart -- Traced Firearm Types by State (Stacked Bar)}
	\label{fig:q2bar}
\end{figure}

\textbf{Insights}: Pistols dominate trace counts in most states, particularly
in the Northeast and Midwest. Rifles show a higher share in Southern and
Mountain states. The variation is substantial and provides a meaningful
second dimension for the analytical question in Q3\sref{q3}.

\subsection{Q3 -- Prevalence vs.\ Traced Firearm Patterns}

\par
A \textbf{scatter plot} was used to explore the association between the
ownership proxy rate and the rifle trace share across states. Each point
represents one state in a given year. Color encodes the gun prevalence
quintile, allowing the viewer to identify clustering patterns visually.
A trend line and confidence interval were added from the Tableau Analytics
pane.

\begin{figure}[h]
	\centering
	\includegraphics[width=\textwidth]{../figures/q3_scatter.png}
	\caption{Q3 Chart -- Gun Prevalence Proxy vs.\ Rifle Trace Share}
	\label{fig:q3scatter}
\end{figure}

\par
A \textbf{dashboard} integrates the Q1 map, Q2 stacked bar, and Q3 scatter
with a shared year parameter filter, a state click-to-filter dashboard action,
and enhanced tooltips showing the key metrics for each state-year combination.
This brings the three questions together in a single interactive view.

\textbf{Insights}: States with higher estimated gun prevalence tend to show
a higher share of traced rifles relative to pistols. However, the relationship
is not uniform and there is variation within high-prevalence states, suggesting
other structural factors are also at play. This is consistent with the
observational and proxy-based nature of both datasets, and no causal
interpretation is claimed.

\section{Conclusions}

\par
Visualizations reveal the following:

\begin{itemize}
	\item Gun ownership proxy is highest in Southern and Mountain West states
	      and has declined nationally since the mid-1990s [Fig.~\ref{fig:q1map}].
	\item Pistols dominate ATF trace counts in most states, but rifle shares
	      are notably higher in higher-prevalence states [Fig.~\ref{fig:q2bar}].
	\item States with higher estimated household gun prevalence tend to show
	      higher rifle trace shares, though this is associative and not causal
		      [Fig.~\ref{fig:q3scatter}].
	\item The two datasets together provide complementary views: estimated
	      household gun availability and traced crime-gun patterns by
	      state and year.
\end{itemize}
